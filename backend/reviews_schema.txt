                                     Table "public.reviews"
        Column        |            Type             | Collation | Nullable |      Default       
----------------------+-----------------------------+-----------+----------+--------------------
 id                   | uuid                        |           | not null | uuid_generate_v4()
 product_id           | uuid                        |           | not null | 
 user_id              | uuid                        |           | not null | 
 order_id             | uuid                        |           |          | 
 rating               | integer                     |           | not null | 
 title                | character varying(255)      |           |          | 
 comment              | text                        |           |          | 
 is_verified_purchase | boolean                     |           |          | false
 is_approved          | boolean                     |           |          | true
 helpful_count        | integer                     |           |          | 0
 created_at           | timestamp without time zone |           |          | CURRENT_TIMESTAMP
 updated_at           | timestamp without time zone |           |          | CURRENT_TIMESTAMP
Indexes:
    "reviews_pkey" PRIMARY KEY, btree (id)
    "idx_reviews_created_at" btree (created_at)
    "idx_reviews_is_approved" btree (is_approved)
    "idx_reviews_product_id" btree (product_id)
    "idx_reviews_rating" btree (rating)
    "idx_reviews_user_id" btree (user_id)
    "reviews_product_id_user_id_order_id_key" UNIQUE CONSTRAINT, btree (product_id, user_id, order_id)
Check constraints:
    "reviews_rating_check" CHECK (rating >= 1 AND rating <= 5)
Foreign-key constraints:
    "reviews_order_id_fkey" FOREIGN KEY (order_id) REFERENCES orders(id) ON DELETE SET NULL
    "reviews_product_id_fkey" FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
    "reviews_user_id_fkey" FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
Triggers:
    update_reviews_updated_at BEFORE UPDATE ON reviews FOR EACH ROW EXECUTE FUNCTION update_updated_at_column()
